Links to WIREFRAMES and CITATIONS

Navigation Button

Buttons for users to navigate to select, confirm and navigate through pages in the webpage.

CSS Selector:

.button-nav

Sample code:

<a href="" class="button">DEFAULT BUTTON</a>

Rendered element:

DEFAULT BUTTON

In-text link

Hyperlinked text in body text.

CSS Selector:

.in-text

Sample code:

<a href="" class="in-text">HyperSpeed Collection</a>

Rendered element:

HyperSpeed Collection

Submit/Reset Buttons

Form and order submission/reset buttons. Buttons will either submit the form filled by the user or clear all entry fields for re-entry.

CSS Selector:

.button_submit

.button_reset

Sample code:

<a class="button_submit">SUBMIT</a>

<a class="button_reset">RESET</a>

Rendered element:

SUBMIT

RESET

Text Input Field

Input fields where users input their information for filling out forms.

CSS Selector:

.input_field

Sample code:

<input type="text" class="input_field">

Rendered element:

Headings

Headings of main content sections and pages.

CSS Selector:

h2

Sample code:

<h2>The Flash</h2>

Rendered element:

The Flash

Paragraphs

Paragraph and body text of website for content information.

CSS Selector:

p

Sample code:

<p>The Flash super runners provide users immense comfort.</p>

Rendered element:

The Flash super runners provide users immense comfort.

Bulleted Lists

Bullet point list for displaying the specs and features of products.

CSS Selector:

ul

Sample code:

<ul> <li>Speed</li> <li>Battery</li> <li>Size</li> </ul>

Rendered element:

  • Speed
  • Battery
  • Size

Numbered Lists

Numbered point list for displaying steps or instructions for user.

CSS Selector:

ol

Sample code:

<ol> <li>Email</li> <li>Password</li> <li>Question</li> </ol>

Rendered element:

  1. Email
  2. Password
  3. Question
